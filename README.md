# IPL Team Web App

A modern full-stack IPL team web application built using React (Frontend) and Node.js + Express (Backend). This app displays team details, players, match results, live scores, and fan interaction features.

---

## Features

### Frontend

* Hero section with team branding
* Player listing with stats
* Match history and scorecards
* Live match section
* Fan poll system
* Latest news section
* Responsive UI

### Backend

* REST APIs for players, matches, polls
* MongoDB database integration
* Structured routes and models

## Installation

### 1. Clone repository

git clone <your-repo-url>

---

### 2. Setup Frontend

cd client
npm install
npm start

---

### 3. Setup Backend

cd server
npm install
npm run server

## API Routes

* /api/players → Get all players
* /api/matches → Get match data
* /api/poll → Submit / get poll results
